# Plugin Libraries

This folder should contain `godot-lib.aar` which is required to build the plugin.

## How to Get godot-lib.aar

This file is generated by Godot when you export your Android project. You have two options:

### Option 1: Copy from Your Build
```bash
cp ../android/build/libs/debug/godot-lib.template_debug.aar godot-lib.aar
```

### Option 2: Download from Godot Releases
1. Go to https://github.com/godotengine/godot/releases
2. Find your Godot version (4.x)
3. Download `godot-lib.X.X.X.template_release.aar`
4. Rename it to `godot-lib.aar` and place it here

## Note
This file is NOT included in Git because it's 105+ MB and exceeds GitHub's file size limit.
Each developer needs to get their own copy.
